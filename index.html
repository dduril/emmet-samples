Brackets using the Emmet Extension

The markup for this page was created using Emmet.

For starters, this is all that was required to create the HTML structure for this page.

html:5

Typing this into Brackets and hitting the tab key, will result in the following HTML5 template being rendered:

  <!DOCTYPE html>
  <html lang="en">
    <head>
      <meta charset="UTF-8">
      <title>Document</title>
    </head>
    <body>
    
    </body>
  </html>

Once the basic page structure has been generated, additional HTML elements and structures can be quickly generated using Emmet.

There are a quick Emmet shortcuts for generating HTML for headers, divs, paragraphs, lists, tables, etc. But, the real power comes from stringing several of the shortcuts together. For example, the following, somewhat advanced example basically says "I want to generate some HTML that has:

  • A div with an id of 'container'
  • With a h1 with a class of 'header1'
  • With a h2 with a class of 'header2'
  • With a p with 10 words of lorem ipsum
  • With an unordered list
  • Containing five list items, each with a hyperlink
  • And another p of 10 words of lorem ipsum

This would be coded in Emmet like this:

div#container>h1.header1+h2.header2+p>lorem10^ul>li*5>a^^p>lorem10 [tab key]

  <div id="container">
    <h1 class="header1"></h1>
    <h2 class="header2"></h2>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Similique, in!</p>
    <ul>
      <li><a href=""></a></li>
      <li><a href=""></a></li>
      <li><a href=""></a></li>
      <li><a href=""></a></li>
      <li><a href=""></a></li>
    </ul>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Incidunt, dolores.</p>
  </div>

HTML5 Examples

Header

header#header>nav#nav>ul>li*4>a.nav-link

  <header id="header">
    <nav id="nav">
      <ul>
        <li><a href="" class="nav-link"></a></li>
        <li><a href="" class="nav-link"></a></li>
        <li><a href="" class="nav-link"></a></li>
        <li><a href="" class="nav-link"></a></li>
      </ul>
    </nav>
  </header>

Section

section.section>article.article>header.header+p

  <section class="section">
    <article class="article">
      <header class="header"></header>
      <p></p>
    </article>
  </section>

Aside

aside.aside>h2+p

  <aside class="aside">
    <h2></h2>
    <p></p>
  </aside>

Footer

footer#footer>p

  <footer id="footer">
    <p></p>
  </footer>
